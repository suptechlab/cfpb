Set up D3

{% highlight javascript %} d3.select('#data').append('p').text('I\'ve been appended!'); {% endhighlight %}

Binding Data

{% highlight javascript %} var dataset = [1, 2, 3]; d3.select('#data').selectAll('p') .data(dataset) .enter() .append('p') .text('I\'ve been appended!'); {% endhighlight %}

Creating a Bar Chart

{% highlight javascript %} d3.select("#data").selectAll("div") .data(complaintdata) .enter() .append("div") .attr("class", "bar") .style("height", function(d) { var barHeight = d * 5; return barHeight + "px"; }); {% endhighlight %}
