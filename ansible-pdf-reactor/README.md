Role Name

A role to install PDF refactor.

Requirements
------------

PDFReactor requires a license.

Role Variables
--------------

PDF_REACTOR_FILE

PDF_REACTOR_URL

PDF_REACTOR_INSTALL_DIR

Dependencies
------------

None.

Example Playbook
----------------

Here is how to install PDF Reactor onto your machine.

    - hosts: servers
      roles:
         - { role: vccabral.pdf-reactor }
