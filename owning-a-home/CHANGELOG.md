# Change Log
All notable changes to this project will be documented in this file.
This project adheres to [Semantic Versioning](http://semver.org/).

## [2.0.12] – 2018-02-21
### Removed
- Set header and footer on-demand CSS font paths to be absolute
- Update header/footer to latest from cfgov-refresh.

### Removed
- IE7 CSS.

### Changed
- Update cfgov-sheer-templates dependency to 2.1.5 to support cfgov-refresh's
  update to Capital Framework 4.
- Hiding journey links and adjusting process steps margin top.
